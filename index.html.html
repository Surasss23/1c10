1. INTRODUCTION
In order to eliminate fraud incidence, manual records, time consumption to search for records of various entities, this system is developed. With the help of this system, management can easily and efficiently manage the daily activities in a hotel.

1.1 Purpose
The main purpose of Hotel Management System (HMS) is to automate and keep track of every transaction like room reservation, room cancellation, payment refund, user details etc. This system helps in monitoring and security of the data.

1.2 Scope
The user can get to know the details of the hotels and can reserve the rooms directly through mobile (or) through website.
Hotel Management System (HMS) provides the details hotel location, room availability on different dates.

Login: User, it allows only authorized users to access the system.
Room availability: It shows the availability of the rooms.
